# FusCuba Messenger

Простая браузерная версия мессенджера на HTML/CSS/JavaScript с функциями регистрации и авторизации.

## 🚀 Возможности

- ✅ Регистрация и авторизация
- ✅ Профилем пользователя с аватаром
- ✅ Список контактов
- ✅ Отправка и получение сообщений
- ✅ Сохранение истории чатов
- ✅ Автоответы на сообщения
- ✅ Адаптивный дизайн
- ✅ Локальное хранилище (localStorage)

## 📋 Структура проекта

```
.
├── auth.html      - Страница входа и регистрации
├── index.html     - Главная страница мессенджера
├── auth.js        - Логика авторизации
├── messenger.js   - Логика мессенджера
├── style.css      - Стили для всех страниц
└── README.md      - Документация
```

## 🔐 Данные по умолчанию

Для тестирования доступны следующие учетные записи:

| Email | Пароль |
|-------|--------|
| alex@example.com | 123456 |
| maria@example.com | 123456 |
| ivan@example.com | 123456 |

## 🎯 Как использовать

1. Откройте `auth.html` в браузере
2. Введите данные из таблицы выше или создайте новый аккаунт
3. Начните общаться с контактами
4. Все сообщения сохраняются локально

## 🎨 Особенности дизайна

- Современный градиент (фиолетовый) на странице входа
- Чистый и интуитивный интерфейс чата
- Аватары пользователей
- Статусы онлайна
- Время отправки сообщений
- Эмодзи в автоответах
