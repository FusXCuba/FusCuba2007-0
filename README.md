# FusCuba Messenger

Простая браузерная версия мессенджера на HTML/CSS/JavaScript.

## 🚀 Возможности

- Простой и интуитивный интерфейс
- Отправка и получение сообщений
- Список контактов
- Адаптивный дизайн

## 📁 Структура проекта

```
.
├── index.html    - Главная страница
├── style.css     - Стили
├── script.js     - Логика
└── README.md     - Документация
```

## 🔧 Как запустить

1. Откройте файл `index.html` в браузере
2. Начните писать сообщения
